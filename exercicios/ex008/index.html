Principais formatações

Negrito ou destaque:

Para colocar um termino em negrito sem respeitar a semântica, usamos a tag B

Agora respeitando a semântica é necessário colocar a tag STRONG

Itálico ou Ênfase

Para colocar uma tag em itálico, assim como o negrito temos a sem semântica, utilizando a tag I

Agora, utilizando uma tag semâtica, é utilizado a tag EM

Texto marcado

Para marcar um texto, usamos a tag MARK

Quando colocamos outro texto em outro parágrafo, as cores ficam diferentes, por isso usamos a aba style na head
porém isso já é de CSS

Texto grande e texto pequeno

Estamos criando um texto grande e um pequeno

Texto deletado

Podemos colocar um texto como excluído para ser desconsiderado

Texto inserido

Já o inserido, serve para mostrar que ele foi adicionado

Existe também a tag sublinhado, que é a U (não semântica)

Texto sobrescrito

Para colocar coisas flutuando, como o ao quadrado x2

Texto Subscrito

Para colocar coisas abaixo da linha, como a quantidade de um átomo em uma molécula. H2O3
